# 📊 Customer Churn Analysis & Retention Strategy  

## 📌 Description  
Customer churn analysis using SQL and Power BI to identify key drivers of customer attrition and revenue loss. Highlights high-risk segments based on tenure, contract type, and services. Includes an interactive dashboard and actionable insights to improve retention and customer lifetime value.  

---

## 🚀 Project Overview  
This project analyzes customer churn to uncover patterns and factors responsible for customer attrition. The goal is to help businesses reduce churn, improve retention, and maximize revenue using data-driven insights.  

---

## 🎯 Business Problem  
- High customer churn leading to revenue loss  
- Lack of clarity on why customers leave  
- No targeted strategy for high-risk customers  
- Inefficient use of customer data  

---

## 🛠️ Tech Stack  
- SQL  
- Power BI  
- Data Analysis  

---

## 📂 Project Workflow  

### 1. Data Cleaning  
- Handled missing values  
- Removed duplicates  
- Ensured data consistency  

### 2. Feature Engineering  
- Created churn flag (0/1)  
- Segmented customers by tenure  
- Calculated average revenue per user  
- Integrated external population data  

### 3. Churn Analysis  
- By contract type  
- By tenure  
- By payment method  
- By internet service  

### 4. Revenue Analysis  
- Total revenue vs churned revenue  
- High-value customer identification  
- Revenue contribution analysis  

### 5. Customer Segmentation  
- Revenue-based segmentation  
- High-risk customer identification  

### 6. Advanced Analysis  
- Multi-factor churn analysis  
- Identification of churn-prone customer profiles  

---

## 🔍 Key Insights  
- 27% overall churn rate  
- 42% churn within first 6 months  
- ~$1.7M revenue lost to competitors  
- Month-to-month contracts have highest churn  
- Lack of tech support increases churn  
- High-value customers also at risk

## 💡 Business Recommendations  
- Promote long-term contracts  
- Improve customer support services  
- Target early-stage customers  
- Offer better pricing and promotions  
- Focus on retaining high-value customers

This project demonstrates how data analytics can be used to identify churn drivers and improve business decision-making. The insights enable targeted strategies to enhance customer retention and increase profitability.
